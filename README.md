## Deskripsi Project
Project ini merupakan aplikasi Web3 sederhana yang dibuat untuk memenuhi UAS mata kuliah Pemrograman Web.  
Aplikasi ini memungkinkan user untuk melakukan **donasi menggunakan cryptocurrency (ETH)** melalui smart contract yang terhubung dengan **MetaMask**.

Project ini menggabungkan konsep **frontend, backend, dan blockchain** dalam satu sistem terintegrasi.

---

## Tujuan Pembuatan
- Menerapkan konsep Web3 dalam aplikasi web
- Memahami integrasi frontend dengan MetaMask
- Mengimplementasikan smart contract menggunakan Solidity
- Menggunakan GitHub sebagai version control project

---

## Teknologi yang Digunakan
- **Frontend** : React.js
- **Backend** : Node.js & Express
- **Blockchain** : Ethereum
- **Smart Contract** : Solidity
- **Wallet** : MetaMask
- **Version Control** : Git & GitHub
